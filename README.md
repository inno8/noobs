# Noobs Project

A collection of beginner coding exercises to learn Python, JavaScript, HTML, and CSS.

## Files
- `todo.py` - Simple todo app
- `calculator.js` - Basic calculator  
- `api.py` - Flask user API
- `user_manager.py` - User registration system
- `shopping_cart.js` - Shopping cart
- `utils.py` - Utility functions
- `index.html` - Landing page
- `style.css` - Stylesheet
